WEB EQ 使い方ガイド

はじめに： 音源の入力方法は3種類あります。Load File（ファイル再生）、audio interface（オーディオIF入力）、Monitor（PCシステム音声）。いずれか1つだけが同時に動作します。

音源の選び方

  • Load File — スマホ・PC内の音楽ファイル（MP3 / M4A / WAV など）を読み込みます。ファイル選択後、Play で再生開始、Stop で停止します。
  • ↻（更新） — 接続中の入力デバイス一覧を更新します。初回はブラウザのマイク許可が必要です。
  • Audio input… — オーディオインターフェースなど、使用する入力デバイスを選びます。
  • audio interface — 選んだ入力から楽器などのライブ音をリアルタイムで EQ に通します。接続中は Disconnect 表示になります。ヘッドホン使用を推奨します。
  • Monitor — PC のシステム音声（ブラウザ・DAW など）をキャプチャして EQ に通します。画面共有ダイアログで「システム音声を共有」を有効にしてください（PC・Chrome / Edge 向け）。

プリセット（Preset）

上部の Preset ドロップダウンから、あらかじめ用意された EQ 設定を読み込めます。

  • Flat — フラット（基準）
  • Vocal Presence — ボーカルを前に出す
  • Tight Bass — タイトな低音
  • Mix Polish — ミックス全体を整える
  • Air Lift — 高音・空気感を強調
  • Deep Cut Test / Telephone Check — テスト用の極端な設定

EQ グラフ（周波数レスポンス）

  • 青い曲線が、現在の EQ 設定による周波数特性です。
  • 色付きの丸（CH 1〜4）をドラッグして、周波数（左右）とゲイン（上下）を調整できます。
  • グラフをクリックして選択後、矢印キーで微調整。Shift + 矢印キーで大きく移動します。
  • 再生中・ライブ入力中は、波形（緑）とスペクトラム（青バー）がリアルタイム表示されます。

4バンド EQ（CH 1〜4）

  • トグルスイッチ — 各バンドの ON / OFF
  • MODE — High Pass / Low Shelf / Peaking / High Shelf / Low Pass
  • FREQ — 中心周波数（20 Hz 〜 20 kHz）
  • GAIN — ブースト / カット量（-30 〜 +30 dB）
  • Q — バンドの鋭さ（値が大きいほど狭い帯域）

出力コントロール（右パネル）

  • OUTPUT メーター — 出力レベルを表示。赤くなったらクリップ（音割れ）の可能性があります。
  • NORMALIZE — 音量を自動で一定に保つ機能。ON にすると入力レベルのばらつきを抑えます。
  • SENSITIVITY — EQ 操作の感度。値を上げるとスライダー・グラフ操作の効きが強くなります。
  • LIMITER — 出力の上限を制限して音割れを防ぎます。Threshold で作動レベルを調整。
  • GAIN — 最終出力音量（-24 〜 +12 dB）

⏻ 電源ボタン（Bypass）

EQ 処理をバイパスし、音源をそのまま出力します。A/B 比較（EQ あり / なし）に使えます。OFF 状態ではグラフの曲線がグレー表示になります。

おすすめの使い方

  • スマホ — Load File で曲を読み込み → Play → プリセットや EQ で調整
  • 楽器・マイク — オーディオ IF を接続 → ↻ でデバイス更新 → 入力選択 → audio interface
  • PC の音 — Monitor でシステム音声をキャプチャ → EQ で調整
